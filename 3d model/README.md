# TitPi — 3D Printed Case

This case holds the Raspberry Pi Zero and the IMX500 AI Camera together on a windowsill, pointing through the glass at the feeder.

### Original description

> This ultra-compact case is designed for the Raspberry Pi Zero and the Camera Module V3/V2, with built-in support for tripod mounting.
>
> To assemble the case, you will need:
> - Raspberry Pi Zero
> - Raspberry Pi Camera Module V3/V2
> - Short camera cable (approx. 3.8 cm)
> - A few small screws
>
> By default, the GPIO pins are fully enclosed for a cleaner look and protection. If you need access to the GPIO, simply cut out the top section above the pins using a utility knife or carpet cutter — the case is designed to make this modification easy and clean.

## Notes for TitPi

- The GPIO cutout modification is useful if you want to attach any additional peripherals
- Printed in PLA at standard settings, costs ~$2 in filament
- The tripod mount is handy for angling the camera precisely at the feeder
